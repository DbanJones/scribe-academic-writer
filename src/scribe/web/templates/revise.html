Source document required

Should be a substantive draft with headings and citations (1,500+ words). Scribe will reject a default scaffold outline. To write from an outline, use New project.

Style optional

Uses the unified academic writing rules (Allwood six-element structure, Williams's characters/actions, Pinker's classic style, Sword's zombie nouns, Amazon's weasel-word discipline). Always on regardless.

What Scribe will and won't change

Preserved exactly

  • Every citation (inline and in the reference list)
  • Every number, percentage, range, unit, and date
  • Every figure and table reference
  • Your argument, framework, and terminology
  • Section structure and heading hierarchy

Sharpened

  • Zombie nouns and empty verbs → concrete action
  • Clutter phrases → concise alternatives
  • Weasel words and unwarranted certainty
  • Sentence rhythm, topic/stress positions, parallel lists
  • Transitions between sections (light smoothing only)
