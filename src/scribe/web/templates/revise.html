Revise an existing draft

Upload a completed draft (1,500+ words, with headings and citations) and Scribe will audit it against the academic writing rules, then revise it section by section. All citations, numbers, data, figures, and claims are preserved exactly. The structure and argument stay yours; only the prose is improved.

This is different from "New project": generation writes a document from scratch using an outline. Revision edits an existing document in place. If you have a completed draft, use this.

Source document Required

Markdown, Word (.docx/.doc), or plain text. Should be a substantive draft with headings and citations, not an outline. Default scaffold outlines are rejected.

Output folder

A new folder will be created if it doesn't exist. Outputs: revised.md and .scribe/audit.md.

Style guide Optional

Leave blank to use the built-in academic style and unified writing rules (Allwood, Williams, Pinker, Sword, Schimel, etc.).
